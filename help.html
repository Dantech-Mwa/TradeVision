Help & FAQ

How do I add a symbol to watchlist?

Go to Screener → click "+ Watch" → it saves to Watchlist.

Is this free?

Yes. 100% open source. No login. No ads. Host yourself.

Can I use real data?

Yes! Replace mock data in `chart.html` with Polygon, Alpha Vantage, or CoinGecko API.

How to deploy?

Fork repo → Settings → Pages → `main` branch → live in 60s.
